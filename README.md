# IoTFi-Hardware

IoT-Fi is a compact low power IoT enabled device having the capability of GSM modules (SIM868/SIM7600G-H) for providing network connectivity via using sim card and having the power of RP2040 chip. It also consists of an LCD 1.14‚Äù display, ESP32-C3, a 3.5mm jack for audio I/p - o/p, SD card and SIM card slot.
